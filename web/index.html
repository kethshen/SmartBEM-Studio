Welcome to SmartBEM Studio

💡 What is SmartBEM?

Ever wanted to simulate a building's energy use without learning complicated software? SmartBEM Studio makes it easy!

Lightweight & Free
Open source. Runs locally on any standard laptop using a web browser and a simple code editor.
NLP Translation
Describe your building layout in plain English, and our workflow will automatically generate the 3D model and simulation files.
Colab-Powered
All heavy processing (Ollama LLM and EnergyPlus engine) runs entirely on Google Colab, not on your laptop.

✨ AI Building Simulator

Describe your building's architecture and thermal configurations in plain English. Our natural language processing workflow translates your text description into a structured EnergyPlus model using local Ollama LLMs. It then runs the industry-standard EnergyPlus simulation engine to generate detailed thermal performance profiles, load calculations, and an interactive 3D geometry model.
