Project Information

"Intelligent HVAC Modeling, Estimation & Control"

This platform bridges the gap between theoretical modeling and practical HVAC control by establishing an AI-assisted framework for EnergyPlus simulation. Users describe buildings in natural language, and a two-pass Agentic RAG pipeline (powered by a local Ollama gemma3:12b model autonomously generates valid IDF files, executes thermal simulations on a Google Colab GPU, and returns time-series result plots to a web dashboard.
