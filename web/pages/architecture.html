System Architecture

Technical overview of the 5-Layer SmartHVAC Studio platform.

High-Level Overview

SmartHVAC Studio integrates a modern web interface with cloud coordination and offline simulation engines.

Components

  • Web Interface (Layer 1): NLP input, simulation config & result dashboard.
  • Cloud Coordination (Layer 2): Firebase Firestore & Storage.
  • Backend Worker (Layer 3): Google Colab GPU runtime polling for jobs.
  • Agentic AI Pipeline (Layer 4): Two-pass RAG with local Ollama gemma3:4b.
  • Simulation Engine (Layer 5): EnergyPlus execution & post-processing.

Design Principles

Separation of Concerns

Interface decoupled from computation via Firebase.

Physics + AI

EnergyPlus accuracy with LLM ease-of-use.

Academic Focus

Transparent data flow for research validation.

Template + Injection

AI outputs structured JSON; all IDF syntax is handled deterministically via Base.idf placeholders.
